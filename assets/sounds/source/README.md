# Fuentes y pruebas de sonido

Guarda aquí archivos de trabajo para efectos de sonido:

```text
assets/sounds/source/
```

Ejemplos:

```text
laser-shot-v2.wav
boss-entry-rumble.wav
thruster-loop-test.wav
enemy-impact-alt.wav
```

Los archivos de esta carpeta no son cargados por el juego hasta que se copien o exporten con el nombre esperado en `assets/sounds/`.
